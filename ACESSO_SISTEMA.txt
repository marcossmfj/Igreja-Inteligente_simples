====================================================
COMO O SISTEMA FUNCIONA (FLUXO DO USUÁRIO)
====================================================

1. CADASTRO: O dono do sistema cria o usuário no Supabase.
2. PRIMEIRO ACESSO: O usuário loga no sistema e, se não tiver 
   uma igreja, o sistema pede para ele criar uma (Multi-tenant).
3. CONFIGURAÇÃO: A igreja cadastra seus 'Cargos' (ex: Pastor, 
   Membro) e suas 'Skills' (ex: Guitarrista, Pregador).
4. MEMBROS: Cadastra as pessoas e vincula às suas habilidades.
5. ESCALA: Ao criar uma escala, o sistema filtra apenas as 
   pessoas que têm a habilidade necessária para o cargo, 
   evitando erros de agendamento.
6. VISITANTES: Registra visitantes e gera um link para mandar 
   mensagem de boas-vindas via WhatsApp com um clique.
